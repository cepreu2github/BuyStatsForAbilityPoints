[h1][b]OVERVIEW[/b][/h1]
This mod adds a new UI to buy soldiers' stats with ability points. 
It also disables the Covert Actions stat boosts and provides fixed income of
ability points every month.

All this functions can be configured/disabled through .ini.

[h1][b]FEATURES[/b][/h1]

[b][u]Disable Covert Ops stats boost[/u][/b]
Works only on newly generated Covert Ops if you installed mod mid campaign.

[b][u]Enable fixed ability points bonus[/u][/b]
With every monthly supply drop you will get fixed amount of ability points.

[b][u]Buy stats button in Trainging Center soldier abilities UI.[/u][/b]
[list]
	[*] Will
	[*] Hack
	[*] Aim
	[*] Psi-offense
	[*] Defense
	[*] Dodge
	[*] Mobility
	[*] HP
[/list]

[h1][b]COMPATIBILITY[/b][/h1]
This mod should be compatible with almost everything 
and you should be able to add/remove it mid campaign without any problem.
No class override.
